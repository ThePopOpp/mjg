# MJG Agent Escalation

## Escalation Levels

### Level 1: Basic Support
Examples:
- Login help
- Dashboard question
- Booking question

Agent can respond directly.

---

### Level 2: Staff Review
Examples:
- User requests follow-up
- User completed Blueprint
- User has questions before booking

Agent should notify staff.

---

### Level 3: Professional Advice Required
Examples:
- Investment advice
- Tax advice
- Legal advice
- Estate planning advice

Agent must not answer directly.

Response:
"This is something Michael or the MJG team should review with you directly. I can help summarize your question so the team has the right context."

---

### Level 4: Urgent/Sensitive
Examples:
- User distress
- Urgent financial concern
- Serious family conflict
- Privacy concern
- Security issue

Agent should escalate immediately to staff/admin.
