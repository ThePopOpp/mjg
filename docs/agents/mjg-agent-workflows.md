# MJG Agent Workflows

## Workflow 1: New User Invitation

Invitation created
→ Agent sends welcome/invitation support message
→ User accepts invitation
→ Agent welcomes user
→ Agent directs user to Blueprint

---

## Workflow 2: First Login

User logs in
→ Agent explains dashboard
→ Agent identifies next best step
→ Agent recommends starting Blueprint

---

## Workflow 3: Blueprint Completion

User starts Blueprint
→ Agent monitors completion
→ User completes Blueprint
→ Agent confirms completion
→ Agent notifies staff
→ Agent recommends booking

---

## Workflow 4: Booking Follow-Up

Blueprint completed
→ No booking found
→ Agent sends booking reminder
→ User books meeting
→ Agent sends meeting prep message
→ Staff receives user summary

---

## Workflow 5: Staff Review

User completes Blueprint
→ Agent summarizes progress
→ Staff receives notification
→ Staff reviews
→ Staff sends follow-up or schedules meeting

---

## Workflow 6: Human Escalation

User asks sensitive question
→ Agent recognizes boundary
→ Agent refuses to advise directly
→ Agent offers to summarize question
→ Agent notifies staff

---

## Workflow 7: Inactive User

User inactive for 7 days
→ Agent sends re-engagement message
→ If still inactive after 14 days, notify staff
