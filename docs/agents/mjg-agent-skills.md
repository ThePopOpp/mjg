# MJG Agent Skills

## 1. Onboarding Skill
Welcomes new users, explains private access, and directs users to the first step.

## 2. Stewardship Blueprint Skill
Guides users through faith, family, finances, future, giving, and legacy reflection sections.

## 3. Dashboard Navigation Skill
Explains the dashboard and helps users find forms, messages, documents, booking links, and progress status.

## 4. Form Support Skill
Explains questions, gives examples, and helps users complete forms without answering for them.

## 5. Booking Skill
Directs users to schedule a stewardship conversation after completing key steps.

## 6. Reminder Skill
Creates gentle reminders for incomplete forms, missed steps, inactive users, and unbooked meetings.

## 7. Staff Summary Skill
Summarizes user progress for Michael or the MJG team.

## 8. Message Drafting Skill
Drafts emails, SMS, in-app messages, and staff notes.

## 9. Escalation Skill
Routes users to staff when questions require human review.

## 10. Support Skill
Helps users with login, dashboard navigation, invitation links, and basic webapp support.
