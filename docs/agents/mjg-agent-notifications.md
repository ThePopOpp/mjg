# MJG Agent Notifications

## Notification Channels

### In-App
Used for:
- Welcome messages
- Dashboard reminders
- Blueprint prompts
- Booking prompts

### Email
Used for:
- Invitations
- Follow-ups
- Blueprint reminders
- Staff summaries

### SMS
Used only with opt-in.

Used for:
- Appointment reminders
- Important reminders
- Short confirmation messages

### Staff Notifications
Used for:
- Completed Blueprint
- User requested help
- User inactive
- Sensitive question asked
- Booking missing

## Notification Tone

All notifications should be:
- Warm
- Brief
- Respectful
- Helpful
- Non-salesy
- Faith-aligned when appropriate
