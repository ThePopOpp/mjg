# MJG Agent Owner Outline

The MJG AI Agent is a private digital assistant inside the MJG WebApp.

It helps users:
- Accept their invitation
- Understand the private experience
- Complete the Stewardship Blueprint
- Prepare for a stewardship conversation
- Book a meeting
- Stay engaged

It helps staff:
- See who needs follow-up
- Review completed Blueprints
- Draft messages
- Monitor inactive users
- Receive alerts when a user needs help

The agent does not give financial, legal, tax, investment, or estate planning advice.
